Simulador de un cajero automático

Mi proyecto para el curso es realizar un simulador de cajero automático, este contara con las opciones básicas de un cajero:
-Ingresar con usuario y nip
-Ver saldo
-Retirar efectivo
-Ingresar efectivo
-Ver movimientos
-Hacer transferncias
-Pagar servicios
-Hacer una recarga de tiempo aire
-Salir
Para esto también voy a simular la cantidad de dinero disponible y tal ves el tanto de billetes que contendrá el cajero para manejarlo de una manera mas realista.
Me gustaría simular varias cuentas, ya que actualmete solo uso una y si quiero cambiar hay que recargar
